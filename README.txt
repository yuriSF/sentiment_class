This is a sentiment classification project.

file_ops: functions for file operations
match_ids.py, match_labels.py, convert_labels.py: data pre-processing
train_glove.py: creates a training model
vectorize.py: functions for model training
params.py: parameter estimation with grid search
classify.py: classifies target data
measure.py: measures precision, recall and accuracy of classification
results/ : output of preprocessing and classification
